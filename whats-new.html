chrome의 새로운 기능

보안

자동 완성으로 빠르고 안전하게 결제하기

카드 보안 코드를 직접 추가하는 대신, 기기를 잠금 해제할 때처럼 지문 스캔, 얼굴 스캔 또는 화면 잠금 PIN 등의 방식을 통해 결제 카드 세부정보를 모두 입력하세요.

  1. 설정 > 자동 완성 및 비밀번호 > 결제 수단으로 이동합니다.
  2. "자동 완성으로 결제할 때마다 수동으로 인증"을 사용 설정합니다.

검색

Google 렌즈로 모든 페이지의 모든 내용 검색

Chrome의 Google 렌즈를 사용하면 화면에서 본 내용을 무엇이든 쉽게 검색할 수 있습니다. 시청 중인 동영상, 라이브 스트림의 슬라이드, 웹페이지의 이미지 안에 있는 항목을 검색해 보세요. 현재 있는 탭에서 Google 렌즈가 바로 정답을 알려 주므로 새로 찾은 정보를 가지고 계속 탐색할 수 있습니다.

  1. Chrome 메뉴를 엽니다.
  2. Google 렌즈로 검색을 선택합니다.
  3. 페이지에서 아무 곳이나 클릭한 다음 드래그하여 페이지에 있는 모든 것을 선택합니다.
  4. 측면 패널에 있는 검색창에 입력하여 답변을 조정합니다.
  5. 더 빠르게 액세스하려면 측면 패널에서 고정을 클릭하여 아이콘을 툴바에 추가합니다.
